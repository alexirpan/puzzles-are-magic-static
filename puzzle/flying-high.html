Flying High

View Solution


View Stats

Storms are popping up above skyscrapers all over Equestria! This looks like a job for Rainbow Dash.

In each city, you'll need to determine the locations of the skyscrapers as well as the storms. You know that:

  • Each city contains skyscrapers with heights 1-5.
  • Every row and column must contain exactly 5 skyscrapers, one of each height.
  • Some of the skyscrapers have storms on them. There are exactly 5 storms in each city, all on skyscrapers with distinct heights (don't ask how you know this).

Additionally, some scouts right outside the cities have given us helpful reports. Every report contains two parts:

  • The number tells you how many skyscrapers can be seen looking from that direction, where taller skyscrapers block the view of shorter ones. This includes skyscrapers with storms on them.

Some cities will have twists which bend the above rules, but if a rule isn't explicitly called out as different, you can assume that it still applies.
